## 🛍️ 功能架构
### 用户端功能
| 模块          | 核心能力                                                                 |
|---------------|--------------------------------------------------------------------------|
| ​**商品中心**  | 支持多级分类(3级类目体系)、商品详情页SPU/SKU分离展示|
| ​**搜索系统**  | Elasticsearch实现语义搜索+多维度过滤(价格/品牌/评分)              |
| ​**交易流程**  | 购物车持久化、优惠券计算引擎、多支付渠道集成(微信/支付宝/银联)           |
| ​**会员体系**  | 成长值累计规则、VIP等级特权、积分兑换商城                                |
